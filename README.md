Zemscripten webgpu triangle example. This zemscripten fork sets emscripten version to 4.0.18. with no additional changes

Staging/storage buffer and bindings example with a compute pipeline

```
git clone git@github.com:ppatt/zemscripten-webgpu-triangle.git
cd zemscripten-webgpu-triangle.git
git submodule update --init --recursive
```
